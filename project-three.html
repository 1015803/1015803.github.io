Subwoofer Repurposing

Samsung uses proprietary subwoofers that can only be used with samsung sound bars. Sadly I lost the peice that connects the subwoofer and soundbar. To fix this I have been splicing directly into hardware components and replacing the proprietary circitry so that when hooked up to an audio controller it will be able to be used with any set of speakers.
